# (void) get_stashes({callback})

Get an array of all stash objects from the Sensu API

Examples:

Using the Tousen namespace, get an array of all stashes from the Sensu API 

Tousen = require 'tousen'
sensu_api = new Tousen url: "http://sensu.example.com:4567"
sensu_api.stashes.get_stashes callback: (err, res) ->
  stashes = res

Using the Tousen namespace, create a stash with path "silence/test_client", content containing a reason, and an expiry of one hour
